# 댓글 문제 디버깅 가이드

게임 대기 화면에서 댓글이 안 써지는 문제를 확인하기 위한 체크리스트입니다.

## 1. 브라우저 개발자 도구 확인 (F12)

### Console 탭
- 에러 메시지가 있는지 확인
- `currentUser`가 null인지 확인하려면 콘솔에 입력:
  ```javascript
  console.log(window.currentUser)
  ```

### Network 탭
- 댓글을 입력하고 전송할 때 `/api/quiz/.../comment` 요청이 가는지 확인
- 요청이 실패하면 Status 코드와 Response 확인

## 2. 화면에서 확인

### 댓글 입력란이 보이는가?
- "퀴즈 댓글" 섹션이 왼쪽에 보여야 함
- 입력 필드와 전송 버튼(Enter 아이콘)이 보여야 함

### 댓글 목록이 로드되는가?
- "댓글을 불러오는 중..." 메시지가 잠깐 보였는가?
- 댓글이 없으면 "아직 댓글이 없습니다." 메시지가 보여야 함

### 로그인 상태 확인
- 오른쪽 위에 프로필이 보이는가?
- 로그아웃 상태라면 댓글을 쓸 수 없음 (로그인 필요)

## 3. 증상별 해결 방법

### 증상 A: 댓글 섹션 자체가 안 보임
→ HTML 로딩 문제 또는 CSS 문제

### 증상 B: 댓글 섹션은 보이지만 입력란이 비활성화됨
→ 로그인 안 되어있거나 초기화 실패

### 증상 C: 입력하고 엔터를 눌러도 아무 반응 없음
→ 이벤트 리스너 등록 실패 또는 API 호출 실패

### 증상 D: 에러 메시지가 나옴
→ 콘솔의 에러 메시지 확인 필요

## 4. 임시 해결 방법

1. 페이지 새로고침 (Ctrl + F5)
2. 브라우저 캐시 삭제
3. 로그아웃 후 다시 로그인
4. 다른 브라우저에서 시도
